# 함수
입력값을 집어넣으면 출력값을 도출해내는 상자박스
- 접근제어자 리턴타입 함수명(파라미터)
- `return` : (반환값)
```java
public int sum(int x, int y){
    return x + y; 
}
```

## Call by Value
파라미터를 Primitive Type으로 설정한 경우
```java
public int sum(int x, int y){
    return x + y;
}
```
