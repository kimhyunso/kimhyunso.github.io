# JVM (Java Virtual Machine)
자바 가상머신
## 가비지 컬렉터
자바는 사용하지 않는 변수 및 함수들을 가비지 컬렉터가 알아서 관리함
- 쓰레기 수집가

## static 영역 (=method 영역)
함수 또는 `static`으로 설정된 변수들이 저장되는 영역

## heap 영역
참조형 데이터(Reference Type), `new` 키워드를 통해 생성되는 인스턴스들이 저장되는 영역

## stack 영역
Primitive Type이 저장되는 영역
