- ## **Rename Refactor : Shift + F6**
- ## 잘라내기 : Ctrl + x
- ## 복제하기 : Ctrl + d
- ## 들여쓰기 줄이기 : Shift + Tab
- ## 클래스 자동으로 만들기 : Alt + Enter
- ## getter, constructor 등 자동 만들기 : Alt + Insert
- ## Run : Ctrl + F5
- ## 상수 만들기 : Ctrl + Alt + c
- ## 같은 폴더 안에 새로운 class 만들기 : Ctrl + Alt + Insert
- ## **코드포맷 : Ctrl + Shift + Alt + L**
- ## 멀티라인 : Ctrl + Ctrl + 방향키아래
