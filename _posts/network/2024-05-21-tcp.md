# 패킷
라우터 경로 최적화를 위해 데이터를 쪼갠 것
## 수신
데이터를 수신 시, 데이터가 무작위로 오는 것을 방지하기 위해 패킷에 번호를 부여한다.

수신 측은 패킷 번호를 통해 잃어버린 패킷 데이터를 추적하고, 무작위로 온 패킷을 재조립한다.


# TCP

## TCP 3 Way Handshake
총 3개의 패킷이 생성된다. ACK 패킷 수신 후 데이터 수신을 받을 수 있다 판단하여 데이터 전송
